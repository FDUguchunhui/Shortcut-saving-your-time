# Tag
- switch between tag

Forward: 
`ctrl + tab`

Backward: 
`ctrl + shift + tab`

close current tag
`ctrl + w`

create new tag:
`ctrl + T`

restore previous closed tag
`ctrl + shift + T`

switch back to previous tag history
`alt + left_arrow`


switch forward to previous tag history
`alt + right_arrow`


# Windows
create a new windows
`ctrl + N`

close current windows
`alt + F4` or `ctrl + shift + w`
